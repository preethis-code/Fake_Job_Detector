# 🔍 Fake Job Detector

An AI-powered web application that detects fraudulent job and internship postings using Machine Learning and Natural Language Processing (NLP).

## Features

- Detects fake job postings
- Uses TF-IDF for text vectorization
- Logistic Regression model
- Confidence score for predictions
- Risk level assessment
- Streamlit web interface

## Tech Stack

- Python
- Scikit-Learn
- Pandas
- Streamlit
- NLP (TF-IDF)

## Dataset

Kaggle Fake Job Postings Dataset

## Model Performance

- Accuracy: 96%
- Fake Job Recall: 74%

## How to Run

```bash
streamlit run app.py
```

## Project Demo

Paste a job description and the model predicts whether it is Real or Fake.
